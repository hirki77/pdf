Comment est ce que j’ai crée mon site ? - 100% IA (ChatGPT)

Ce site a été créé à 99% avec de l'intelligence artificielle. Aujourd'hui, il est simple de créer son site avec de l'IA.

Prérequis
  • Un compte Infomaniak avec un nom de domaine (voir le lien de l'offre plus bas)
  • Un compte ChatGPT basique
  • Une connexion Internet
  • Un minimum de maîtrise de l'ordinateur et de ses outils

Offre Infomaniak : https://www.infomaniak.com/fr/domaines

Je savais exactement quel type de site je voulais. Je souhaitais que mes informations de contact soient affichées avec des boutons pour rediriger vers mes différents projets. Je voulais également que l'heure soit affichée.

Je désirais un fond avec des blocs transparents. Il est important de savoir ce que l'on veut et de faire un croquis sur une feuille blanche pour avoir une idée de la structure du site.

Ensuite, j'ai décrit mes souhaits à ChatGPT. Il m'a rédigé le code HTML du site web. Certains détails ne me convenaient pas, alors je lui ai demandé de retravailler le code avec les modifications souhaitées. Par exemple, au survol de la souris, les boutons devenaient jaune foncé alors qu'ils étaient déjà de cette couleur. Je lui ai donc demandé de changer la couleur au survol en bleu clair. ChatGPT m'a alors fourni le code avec les modifications nécessaires.

Il faut fournir vos informations personnelles à ChatGPT pour qu'il puisse les inclure dans le code. Sinon, vous pouvez les ajouter vous-même (attention à ne modifier que les champs nécessaires sans affecter le reste du code, ce qui pourrait le corrompre). ChatGPT inclut automatiquement des champs comme « (Votre Prénom) » ou « (Lien vers votre PDF) ». Sinon donnez-lui directement les informations.

Pour inclure une image d'arrière-plan, rien de plus simple : il suffit de donner le lien vers l'adresse de l'image (attention : 1920x1080).

Mettez en ligne vos fichiers sur votre Google Drive ou tout autre service de stockage en nuage permettant de partager des fichiers via un lien public. Votre compte Infomaniak inclut d'ailleurs un kDrive de 15 Go gratuit.

Pour ajouter un formulaire de contact avec un bouton de message en bas à droite de votre site, j'ai utilisé le service FormButton. Ce service permet d'ouvrir un petit formulaire de contact, à travers lequel les visiteurs peuvent vous envoyer un message qui sera directement envoyé à votre boîte mail. Vous pourrez ensuite leur répondre via l'adresse e-mail qu'ils auront indiquée dans le formulaire. (Créez un compte, puis suivez les instructions. Copiez le code du bouton de formulaire, donnez-le à ChatGPT et demandez-lui d'inclure le code.)
